# Zarigani

## 機能
- Zariganiは、メールボックスから指定された送信元アドレスのメールを取り出し、該当するメールに添付されているイメージファイルを保存する機能を持つ。
- UIは月ベースのカレンダとする。
- カレンダのマスには、マスの日付と一致するメールの送信日時に添付されていたイメージファイルのサムネイルを表情する。
- サムネイルをクリックすると、イメージファイルが別Windowで表示される。
- Zariganiはイメージファイルを保存するデータベースを持つ。
- データベースには、メールのタイトル、メールの送信日時、保存したイメージファイルのファイル名を格納する。
